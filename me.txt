  Deepgram STT · ElevenLabs TTS · Fastify · PostgreSQL · Redis · BullMQ ·
  Docker · Docker Compose · Turborepo · pnpm workspaces · Prisma · WebSockets

What it does:
  An agentic AI voice assistant that users can call via phone (Twilio). The
  system transcribes speech in real-time (Deepgram), reasons and responds
  using Claude with tool-calling capabilities, then speaks back to the user
  (ElevenLabs) — all with sub-second conversational latency.

Architecture Highlights:
  — Agent Kernel: Centralised tool router, Zod-validated input/output schemas,
    immutable AgentContext object, single audited LLM entry point. This is
    production-grade agentic architecture — not a simple chatbot.
  — Voice Pipeline: Twilio Media Streams → WebSocket → Deepgram STT → Claude
    tool-calling loop → ElevenLabs TTS → WebSocket → Twilio. Real-time,
    sub-second latency.
  — Monorepo Architecture: pnpm + Turborepo typed monorepo with packages:
    ai-core, voice-pipeline, db, queues, events, logger, config — clean
    separation of concerns.
  — Infrastructure: Docker Compose orchestrating all services, BullMQ async
    job workers for background tasks, Prisma migrations for DB schema.

Why it matters:
  Demonstrates mastery of the full AI product stack — from LLM agent design
  patterns to real-time voice infrastructure to production DevOps. This is
  the kind of system most senior engineers haven't built.
